# FPS Galeria de Tiro - Unity

## Proyecto Programación de Videojuegos

Este proyecto consiste en el desarrollo de un videojuego FPS estilo Call of Duty desarrollado en Unity.

## Características

- Movimiento FPS
- Disparo con proyectiles
- Sistema de puntuación
- Enemigos
- Munición
- Sistema de vidas
- Dos niveles
- Menu principal
- Pantalla Game Over

## Herramientas utilizadas

- Unity
- C#
- Visual Studio
- GitHub
